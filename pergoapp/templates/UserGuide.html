Guía de Uso

Antes de utilizar el generador, es necesario crear una cuenta de usuario desde esta misma página. Para ello, deberás ingresar un nombre de usuario, una contraseña (y confirmarla), así como completar algunos datos adicionales como el nombre y las siglas de la radio, un correo electrónico de contacto y el estado o región en el que operas.

Una vez creada la cuenta, deberás iniciar una suscripción y realizar el pago a través de PayPal. Con el mismo usuario y contraseña que registres aquí, podrás acceder a todas las funciones del programa.

El nombre de la radio servirá para identificarla fácilmente, mientras que las siglas se utilizarán para nombrar los audios generados. El correo electrónico nos permitirá mantenerte informado sobre actualizaciones importantes, y el estado o región se solicita con fines estadísticos, para conocer en qué zonas se utiliza el programa.

Es importante que configures tu antivirus para que no considere al programa como una amenaza, de manera que funcione correctamente sin interrupciones. Además, se recomienda agregar el programa para que se inicie automáticamente cuando enciendas la máquina. Puedes seguir los pasos de este artículo para hacerlo: Configurar aplicaciones de inicio en Windows.

Configuración Inicial

Antes de comenzar

Antes de usar el programa, es necesario realizar la configuración inicial. En el siguiente video se muestra paso a paso cómo hacerlo.

Recuerda que necesitas contar con un usuario y contraseña que obtuviste al registrarte en la página.

El programa requiere conexión a internet para funcionar correctamente.

Notas

  • Algunos antivirus podrían mostrar una advertencia al intentar ejecutar el archivo .exe del programa. Esta advertencia puede aparecer al presionar el botón para obtener la lista de dispositivos de audio, al probar la grabación de prueba o al reproducir la grabación de prueba. Si esto ocurre, selecciona la opción "permitir" para ejecutar el archivo y asegurar que el programa funcione correctamente.
  • Para poder probar el dispositivo de entrada de audio para la grabación, primero debe completar la configuración de todos los parámetros y guardar los cambios. Cuando seleccione un nuevo dispositivo, deberá guardar nuevamente la configuración.

Agregar y configurar campañas

Una vez finalizada la configuración inicial, pasamos a la configuración de campañas. En el siguiente video te mostraremos paso a paso cómo utilizar el programa.

Nota importante: Algunos antivirus pueden mostrar una advertencia al intentar ejecutar el archivo .exe del programa, lo que podría evitar que la grabación se inicie correctamente justo después de la configuración inicial. Si esto ocurre, cierra y vuelve a abrir el programa. Una vez que la grabación inicie correctamente, no deberías tener más problemas. Si el error persiste, comuníquese con nosotros.
